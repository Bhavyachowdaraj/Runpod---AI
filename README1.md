# Conversational AI using Faster-Whisper, Gemini, Kokoro and Pipecat

## Objective

Build a conversational AI pipeline using:

- Faster-Whisper (Speech-to-Text)
- Gemini 2.5 Flash (LLM)
- Kokoro (Text-to-Speech)
- Pipecat (Pipeline Orchestration)

## Architecture

User Speech
↓
Faster-Whisper
↓
Transcript
↓
Pipecat
↓
Gemini 2.5 Flash
↓
Response Text
↓
Kokoro
↓
Audio Response

## Technologies

- Python
- Google Colab
- Faster-Whisper
- Gemini
- Kokoro
- Pipecat

## Results

Successfully implemented:

- Speech Recognition
- LLM Integration
- Text-to-Speech
- End-to-End Conversational Pipeline
